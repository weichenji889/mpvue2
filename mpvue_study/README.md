# mpvue(开发微信小程序，测试)Demo
## 该项目可以直接导入小程序开发工具使用
开发步骤：
### 1.```vue init mpvue/mpvue-quickstart my-project``` 在命令行，使用该命令在根文件夹下初始化一个项目
### 2.```npm init```  在命令行，使用该命令下载对应的依赖包
### 3.使用小程序开发工具打开项目，即可预览项目效果
